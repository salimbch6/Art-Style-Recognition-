
# Art Style Recognition

This project is a deep learning web application that classifies artworks into different artistic styles using **EfficientNet-B0** and **Flask**.

## Features
- Upload an image or paste an image URL.
- Predicts the art style of the image.
- Modern and intuitive web interface using **TailwindCSS**.
- Pre-trained EfficientNet-B0 model fine-tuned on WikiArt dataset.
